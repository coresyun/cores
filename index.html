FAQ

기능과 이미지 처리 방식, 링크 생성에 대해 자주 묻는 내용을 정리했습니다.

정말 무료인가요?
네. 이 도구 자체는 무료이고, 이미지 호스팅에 사용하는 ImgBB도 무료 API 키로 이용할 수 있어요. 별도의 결제나 구독이 필요하지 않습니다.
어떤 이미지 형식을 지원하나요?
JPG, PNG, GIF, WebP, SVG를 지원합니다. JPG와 PNG는 자동으로 압축되고, GIF와 SVG는 원본 그대로 유지됩니다. PNG의 투명 배경도 보존돼요.
이미지가 외부 서버에 저장되나요?
압축은 브라우저 안에서 처리되지만, 짧은 URL을 만들기 위해 압축된 이미지는 ImgBB 서버에 업로드됩니다. 민감한 이미지는 올리지 않는 것을 권장해요.
API 키는 어디에 저장되나요?
API 키는 이 브라우저의 로컬 저장소에만 보관되며, 다른 곳으로 전송되지 않습니다. 다른 기기나 브라우저에서는 다시 등록해야 해요.
압축 품질은 조절할 수 있나요?
네. 업로드 섹션에서 최대 너비(640 / 1200 / 1920)와 화질(60~95%)을 자유롭게 설정할 수 있어요. 용도에 맞게 파일 크기와 품질의 균형을 맞추세요.
생성된 URL은 얼마나 유지되나요?
ImgBB에 업로드된 이미지는 기본적으로 영구 보관됩니다. 다만 외부 서비스 정책에 따라 달라질 수 있으니 중요한 이미지는 별도 백업을 권장해요.
한 번에 여러 장을 올릴 수 있나요?
네. 여러 이미지를 동시에 끌어다 놓으면 각각 압축·업로드되어 결과 목록에 표시됩니다. 전체 URL을 한 번에 복사할 수도 있어요.
모바일에서도 사용할 수 있나요?
네. 반응형으로 제작되어 모바일 브라우저에서도 이미지를 선택해 업로드하고 링크를 복사할 수 있어요.
업로드가 실패하면 어떻게 하나요?
API 키가 올바른지 먼저 확인해주세요. 키가 정확한데도 실패한다면 잠시 후 다시 시도하거나, 이미지 용량을 줄여보세요.
이미지 호스팅은 ImgBB의 무료 API를 사용합니다. 덕분에 별도 서버 없이도 짧은 이미지 링크를 만들 수 있어요.
